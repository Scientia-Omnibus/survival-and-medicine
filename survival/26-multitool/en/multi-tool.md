> A multi-tool can be useful in everyday life and in survival situations.

## Common Tools

| Tool | Primary Use |
|---|---|
| Knife | Cutting, carving, food preparation |
| Can opener | Opening sealed food containers |
| Screwdriver | Repairing equipment |
| Pliers / pinchers | Gripping, bending, manipulating objects |
| Wire cutter | Cutting wire, cable, small fasteners |
| Wood saw | Cutting branches and wood for shelter |

## Additional Tools

| Tool | Use |
|---|---|
| Bit driver | Driving screw bits |
| Whistle | Signaling |
| Scissors | Cutting fabric, cordage |
| Ferrocerium rod | Starting fires |
| Sharpener | Maintaining blade edges |

## Selection Considerations

- Trade off weight and size against the tool set provided.
- Regular carry and use builds familiarity and confidence.
- Leave it at home only when security restrictions or activities prevent carrying it.
